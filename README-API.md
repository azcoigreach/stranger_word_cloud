wordcloud-api
=============
A FastAPI implentation of [amueller/wordcloud][forked_from]

## Build and Install
Build cython
```
python setup.py build_ext --inplace
```
Install wordcloud
```
pip install . --use-feature=in-tree-build --no-cache-dir
```



[forked_from] https://github.com/amueller/word_cloud
